# Tasks

**Context:** Добавление страницы управления командой и ссылки приглашения коллег из формы проекта с отображением названия и количества участников.

## Main directions
### UI и маршруты
- [ ] Создать страницу `pages/projects/[id]/team.vue` с версткой согласно макету: шапка с кнопкой назад, поле поиска, список карточек участников и кнопка добавления.
- [ ] Обеспечить соответствие стилистике тёмной/светлой темы, переиспользуя существующие utility-классы и токены цветов.
- [ ] Добавить новый маршрут в Nuxt для страницы управления командой и убедиться, что навигация доступна из формы проекта.

### Компоненты и логика
- [ ] Реализовать компонент `ProjectInviteLink` (или аналог) под полем названия проекта, который формирует текст ссылки с именем проекта и количеством коллег, обрабатывает клик и переход.
- [ ] Настроить реактивное обновление текста ссылки при изменении названия проекта и при загрузке счётчика участников.
- [ ] Создать компонент `TeamMemberCard` с пропсами для имени, роли, аватара и иконки действий; обеспечить доступность (aria-label для кнопок).
- [ ] Добавить компонент или composable для поиска по списку участников, фильтрующий отображаемый список в режиме реального времени.

### Данные и интеграции
- [ ] Определить и подключить источник данных для списка участников по `projectId`; при необходимости добавить временный mock/заглушку.
- [ ] Синхронизировать счётчик участников между страницей проекта и страницей команды, обновляя значение при изменении списка.
- [ ] Добавить юнит-тесты для функции форматирования текста ссылки и логики фильтрации участников.

## Supporting tasks
- [ ] Documentation: обновить описание страниц проекта и команды, указав новый маршрут и компонент ссылки.
- [ ] Observability: Not required — reason: клиентская функция без бекенд-метрик; мониторинг не меняется.
- [ ] Code review and PR: подготовить изменения для ревью и описать ключевые сценарии.
