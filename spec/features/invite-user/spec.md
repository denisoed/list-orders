# Приглашение коллег в проект

**Specification:** Добавить страницу управления командой с списком коллег и доступом к приглашению через ссылку из формы создания/редактирования проекта. Ссылка отображается под полем названия проекта и содержит название проекта и количество коллег. На новой странице показывается список участников с возможностью поиска и действий.

## User Stories
- Как менеджер проекта, я хочу из формы создания проекта перейти по ссылке приглашения, чтобы сразу добавить коллег после настройки проекта.
- Как менеджер проекта, я хочу видеть в тексте ссылки имя проекта и текущее количество приглашённых коллег, чтобы понимать статус команды перед переходом.
- Как участник команды, я хочу на странице управления командой находить коллег через поиск и видеть их роли, чтобы быстро ориентироваться в составе команды.

## Main scenarios and rules
- На страницах создания и редактирования проекта под полем названия отображается ссылка «Пригласить коллег (N)» где N — количество приглашённых коллег; название проекта подставляется в текст ссылки или подсказку.
- Переход по ссылке открывает новую страницу «Управление командой» в едином стиле приложения (тёмная тема Telegram WebApp) с заголовком, поиском и списком участников.
- Страница управления командой отображает список текущих участников с именем, ролью и аватаром; данные берутся из API проекта или временного стора.
- Список поддерживает действие «ещё» для каждого участника (иконка «more_vert») для будущих операций; пока без функционала.
- В верхней части страницы есть поле поиска по имени коллеги; фильтрация работает по подстроке без учёта регистра.
- Страница должна корректно работать в светлой и тёмной темах, используя существующие токены цветов.
- Если название проекта отсутствует, ссылка отображает плейсхолдер «проект» вместо имени.
- Количество коллег в ссылке обновляется при изменении состава команды.
- Должен быть предусмотрен переход назад к форме проекта (кнопка стрелки в шапке).

## Non-functional requirements
- UI соответствует примерам и дизайн-токенам Tailwind/Telegram, адаптивен для экранов от 320px.
- Производительность: фильтрация списка выполняется мгновенно на клиенте без дополнительных запросов при локальном списке до 100 участников.
- Доступность: интерактивные элементы должны быть доступны с клавиатуры и иметь aria-атрибуты/подписи.
- Локализация: весь текст на русском языке, поддержка будущей смены языка через словарь.
- Код должен быть покрыт юнит-тестами для форматирования текста ссылки и логики счётчика участников.

## Assumptions
- API для получения списка участников уже доступно или данные временно берутся из mock-хранилища; требуется уточнить точный источник данных.
- Механизм приглашения новых пользователей пока не реализуется; страница служит для навигации и отображения существующих коллег.
- Параметры перехода (id проекта) доступны из роутера и используются для загрузки данных, требуется подтвердить текущий маршрут.
